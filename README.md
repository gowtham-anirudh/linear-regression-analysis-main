# Linear Regression Model for Used Car Prices

This project uses a linear regression model to predict outcomes from structured data using a used car prices dataset.

Working of this project
- Loads the training and testing dataset
- Trains a linear regression model
- Predicts car prices based on features
- Evaluates model performance

Project contents
- `Linear_Regression.ipynb`: Jupyter Notebook with code & explanation
- `train.csv`: Training dataset
- `test.csv`: Testing dataset

Tools & Libraries
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib / Seaborn

How to run
1. Clone this repository  
2. Open `Linear_Regression.ipynb` in Jupyter  
3. Run all cells step-by-step
